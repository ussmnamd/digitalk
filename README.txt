Digitalk Website Project
========================

This is the source code for the Digitalk agency website. 
It is built with Next.js and React, designed to be lightning-fast, fully responsive, and highly optimized for SEO.
This repository contains only the files required by Vercel for building and serving the static/serverless application.

Technology Stack:
-----------------
- Next.js
- React
- Tailwind CSS
- Framer Motion for animations
- Lenis for smooth scrolling

Deployment:
-----------
This project is configured for seamless deployment on Vercel. 
The main branch is directly linked and builds automatically upon receiving pushes.

For any questions about the structure or maintenance, please consult the development team.
